### 3. **Added Controller Methods**

Added 4 new methods to `ReportController.php`:

-   `salesPdf()` - Generate sales report PDF
-   `inventoryPdf()` - Generate inventory report PDF
-   `seniorCitizenPdf()` - Generate SC discount report PDF
-   `pwdPdf()` - Generate PWD discount report PDF

Each method:

-   Accepts same filters as web view
-   Generates descriptive filename with date/period
-   Returns PDF for download
-   Handles errors gracefully

### 4. **Updated Routes** (`routes/web.php`)

Added PDF generation routes:

```php
Route::get('/reports/sales/pdf', [ReportController::class, 'salesPdf'])->name('reports.sales.pdf');
Route::get('/reports/inventory/pdf', [ReportController::class, 'inventoryPdf'])->name('reports.inventory.pdf');
Route::get('/reports/senior-citizen/pdf', [ReportController::class, 'seniorCitizenPdf'])->name('reports.senior-citizen.pdf');
Route::get('/reports/pwd/pdf', [ReportController::class, 'pwdPdf'])->name('reports.pwd.pdf');
```

### 5. **Updated UI** (`resources/views/reports/index.blade.php`)

Replaced "Print Report" button with:

-   **"Download PDF Report"** button
